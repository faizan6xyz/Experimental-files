## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/faizan6xyz/Experimental-files.git
cd Experimental-files

# Install common dependencies
pip install pandas numpy matplotlib seaborn scikit-learn scipy
pip install cryptography requests beautifulsoup4 playwright sqlalchemy
pip install torch torchvision  # for YOLOv5

# Run any script
python AESencryption.py
python datacleaning.py
```

> Some scripts may require additional setup (e.g., MySQL credentials for `connecttomysql.py`, or a running database for SQLAlchemy scripts).
